## 用户确认决策

Windows 与 macOS 继续作为产品目标平台。每个发布候选仅要求在一个选定平台执行完整真实桌面旅程；另一平台仍须通过 Wails 构建门禁，但不重复执行完整旅程。

## 结论

变更未取消 Windows/macOS 产品支持，也未降低双平台构建要求；仅将昂贵的完整桌面旅程集中到一个选定平台。新增的工程骨架 AC 使 TASK-001 可独立测试与关闭。七个审查维度无遗留 P0/P1 问题，可以恢复 STEP 6 执行。
