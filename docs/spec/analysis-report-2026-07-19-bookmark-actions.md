## 一致性审查

- 数据模型：不新增字段；继续使用 Bookmark.categoryId、tagIds、collectionIds 与现有 URL/notes 字段。
- API：不新增外部接口；全部为本地领域命令和现有 Repository 保存。
- 安全：URL 继续使用 HTTP(S) 规范化；删除必须显式确认。
- 性能：选择状态仅保存 ID；批量命令单次遍历书签和主题关系。
- 可访问性：所有入口使用可见英文标签、语义化按钮、checkbox 和 dialog。
